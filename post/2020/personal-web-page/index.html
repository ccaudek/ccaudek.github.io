Come ho creato questa pagina web

Last updated on Jul 15, 2020 2 min read

Queste note sono rivolte sopratutto a me stesso, per ricordarmi la procedura che ho seguito per creare questa pagina web e per modificarne i contenuti (per esempio, aggiungendo un post).

  • Ho creato tutti i contenuti del sito web con blogdown in una cartella che ho chiamato my-website. Per verificare i contenuti del sito web è possibile visionare una versione privata del sito che si genera con RStudio mediante l’istruzione
library("blogdown")
blogdown::serve_site()

Si noti che, all’interno di my-website, ci sarà una cartella chiamata public. Informazioni sull’uso di blogdown si trovano qui oppure qui.

  • Ho creato un repository sul mio account GitHub con il nome ccaudek.github.io (il nome del mio account è ccaudek – è necessario che il nome del repository abbia questa struttura) e ho clonato questo repository sul mio Mac. Per fare questo, una volta scelta la directory appropriata, dal terminale ho utilizzato l’istruzione seguente:
git clone https://github.com/ccaudek/ccaudek.github.io.git

  • Una volta creata la cartella ccaudek.github.io sul mio Mac, ho copiato all’interno di tale cartella tutto il contenuto della cartella my-website/public creata da blogdown – è necessario copiare e incollare il contenuto della cartella public, non la cartella public!

  • A questo punto ho clonato nel repository GitHub ccaudek.github.io tutti i file che si trovano nella corrispondente cartella sul mio Mac:

git status
git add --all
git commit -m "commento"
git push

Dopo qualche minuto il contenuto del sito web sarà online all’indirizzo https://ccaudek.github.io/.

  • Utilizzo la stessa procedura quando modifico un contenuto del sito, per esempio quando aggiungo un post al blog.
